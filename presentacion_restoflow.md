## ¿Qué es RestoFlow?
RestoFlow es una solución SaaS integral diseñada para la gestión moderna de restaurantes. Desde la administración de múltiples sucursales (multi-tenant) hasta la operación diaria en el salón, RestoFlow optimiza cada proceso para maximizar la rentabilidad y mejorar la experiencia del cliente.

## Fortalezas del Sistema

### 1. Seguridad y Aislamiento de Datos (RLS)
Utilizamos **Row Level Security (RLS)** avanzado a nivel de base de datos. Esto garantiza que la información de cada empresa esté totalmente aislada, permitiendo que una sola plataforma sirva a miles de restaurantes con total privacidad.

### 2. Control Antifraude (Prevención de Fugas)
El sistema implementa restricciones jerárquicas críticas. Los meseros pueden gestionar pedidos, pero **no pueden cancelarlos ni resetear mesas** sin supervisión. Esto elimina la "fuga hormiga" de dinero, asegurando que cada orden abierta deba ser facturada o justificada por un administrador.

### 3. Gestión Visual del Salón
El plano de mesas interactivo permite ver el estado del restaurante de un vistazo:
*   **Verde:** Disponible para nuevos clientes.
*   **Amarillo:** Mesa ocupada con pedido pendiente.
*   **Azul:** Mesa atendida y lista para facturar.

### 4. Ciclo de Venta Optimizada

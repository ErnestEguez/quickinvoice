# MANUAL DE INDUCCIÓN: SUPER ADMIN

## 1. Perfil del Super Admin
El Super Administrador es el encargado de gestionar la plataforma a nivel global. Su acceso permite supervisar todas las empresas registradas y el personal administrativo de cada una.

## 2. Funciones Principales

### A. Gestión de Empresas
- **Visualización Global:** Puede ver el listado de todas las empresas que utilizan QuickInvoice.
- **Creación de Empresas:** Registro de nuevos RUCs, nombres comerciales y configuraciones base para cada restaurante.
- **Edición de Datos Maestros:** Modificación de datos fiscales y parámetros de facturación global.

### B. Gestión de Usuarios de Oficina
- El Super Admin crea los usuarios de tipo **Oficina** para cada empresa.
- Estos usuarios serán los administradores locales de cada restaurante.

### C. Monitoreo de Plataforma
- Acceso a logs técnicos y estado de servicios.
- Supervisión del uso de la plataforma por parte de los inquilinos (tenants).

## 3. Navegación en la App
Al ingresar como Super Admin, el sistema redirige automáticamente a la sección de **Administración de Plataforma**:
- **Pestaña Empresas:** Listado y creación de locales.
- **Pestaña Personal:** Gestión de los gerentes (Oficina) de cada local.

---
> **Nota de Seguridad:** El Super Admin tiene permisos de lectura sobre toda la base de datos sin restricciones de empresa. Úselo con responsabilidad.

# MANUAL DE INDUCCIÓN: MESEROS

## 1. Perfil del Mesero
El mesero es el motor operacional del salón. El sistema está optimizado para una atención rápida y eficaz desde tablets o dispositivos móviles.

## 2. Operación Diaria

### A. Gestión de Mesas
1. Ingrese a **"Gestión de Mesas"**.
2. Seleccione una mesa con estado **Verde (Libre)**.
3. El sistema abrirá la interfaz de toma de pedido.

### B. Toma de Pedidos
- Seleccione los productos solicitados por el cliente.
- Confirme el pedido. El sistema enviará automáticamente la comanda a cocina/barra.
- La mesa cambiará a estado **Amarillo (Ocupada)**.

### C. Servicio y Facturación
- Una vez servidos los platos, puede marcar el estado como "Atendido".
- Cuando el cliente solicite la cuenta, notifique al administrador para proceder a la facturación.

## 3. Limitaciones de Seguridad (Importante)
Para garantizar la integridad de la caja:
- **No puede cancelar ítems** de un pedido ya confirmado.
- **No puede resetear mesas** ocupadas.
- Si cometió un error o el cliente desea desistir de un pedido, **debe llamar obligatoriamente al Usuario Oficina** para que autorice y realice la anulación.

---
**Recuerde:** Cada pedido abierto en el salón representa un compromiso financiero. El sistema no permitirá cerrar la jornada si hay mesas ocupadas sin facturar.

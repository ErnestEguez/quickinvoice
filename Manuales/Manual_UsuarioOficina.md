# MANUAL DE INDUCCIÓN: USUARIO OFICINA

## 1. Perfil del Usuario Oficina
Es el administrador del local. Su responsabilidad es configurar el catálogo de productos, gestionar al personal de servicio (meseros/cocina) y supervisar la facturación diaria.

## 2. Responsabilidades Clave

### A. Gestión de Personal (Control de Seguridad)
- **Creación de Meseros:** Registrar nombres, correos y PINs de acceso para su equipo.
- **Auditoría:** Es el único facultado para **cancelar pedidos** o resetear mesas si ocurrió un error en el salón.
- **Seguridad:** El sistema filtra la lista de personal para que un administrador de oficina no se vea a sí mismo en los roles de servicio, manteniendo la interfaz despejada.

### B. Mantenimiento del Catálogo
- Registro de categorías y productos.
- Configuración de precios e impuestos (IVA 15%).
- Gestión de inventarios (Kardex) para controlar el stock de insumos y bebidas.

### C. Facturación SRI y Cierres (Nuevos)
- **Facturación Electrónica:** Al facturar, el sistema valida RUC/Cédula con el SRI.
- **Detección Automática:** Si ingresa la identificación, el sistema completa el nombre automáticamente.
- **Formatos:** El sistema imprime automáticamente el ticket de 80mm tras facturar.
- **Cierre de Caja:** Al final del turno, realice el arqueo en la sección "Cierres". Debe cuadrar el efectivo, tarjetas y depósitos antes de cerrar la sesión.

## 3. Herramientas del Sistema
- **Dashboard:** Estadísticas de ventas del día, pedidos activos y mesas ocupadas.
- **Configuración:** Acceso a toda la gestión de datos locales del restaurante.
- **Kardex:** Verificación de movimientos de entrada y salida de productos.

---
> **Punto Crítico:** Usted es el respaldo de seguridad del mesero. Si un mesero solicita cancelar un pedido, solo usted puede ejecutar esta acción tras verificar el motivo.

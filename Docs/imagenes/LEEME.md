## Cómo tomar las capturas

1. Use **F12** → modo responsive para simular tamaño de pantalla específico, o capture la pantalla completa.
2. Herramienta recomendada: **Snipping Tool** (Win + Shift + S) en Windows.
3. Guarde en formato **PNG** para mejor calidad.
4. Resolución sugerida: mínimo **1280 × 720** píxeles.
